Instructions

Recording player data

To record data first select a tournament and press the Select tournament button. This will then allow the match selection from the dropdown box. Press Select match once a selection has been made. The list of players for that match will then be enabled; from this select the player and the position they played (for ease of reference any player selected from the dropdown menu has their team displayed on the right hand side of the screen).
Once this is done press the Select player button which will now bring up the data collection buttons most of which are initially greyed out. These are position dependent.

To record data about a player press the relevant button when they commit said action. This will record the data to a dataframe ready to be exported to an excel document.
If a player receives two yellow cards or one red card they are ejected from the game and all the buttons to record data become greyed out.

Saving a player

Once a player’s stats for the match press the Save player button which will then export the statistics to the relevant files.
This will then reset all statistics back to zero so that a new player is abled to be recorded.

Player with multiple positions

If a player has multiple positions within one game such as a keeper and a seeker this must be recorded as if each position was a separate player as the program cannot currently account for this.

Example:

Player A is initially a chaser but then becomes a seeker for a single stint before returning as a chaser they should be marked as off pitch whilst they are seeking until they return to the pitch as a chaser.

Making an error

If an error is made whilst recording a players statistics then the reset button should be pressed. This resets all the currently recorded statistics to 0. Therefore it is highly encouraged to not make a mistake as pressing this button means that the players data for the selected match will have to be recorded again from the beginning.

Add new match

To access the add match screen go to the file menu in the top left hand corner of the window. This will bring up a menu with the Add new match button. Clicking this will bring up a new window where a new match may be added to the database.

Adding a new match

To add a new match fill in the name of the tournament, season it took place in and the team names in the relevant boxes. Then type a name of a player in the first team (team 1) into the player name box and press the add player button. This will add them to the top player list box on the right-hand side of the window. Keep adding players until the roster is complete and repeat this for the players on the other team.

Removing a player from a roster

To remove a player from either teams roster select the players name in the player list box and press Remove player. This player will now disappear from the box and is removed from the roster.

Saving match

To save a match press the save match button. If every field is not filled in an error message will appear and the match will not be saved.
If the match has saved then all the fields will be cleared.
If a match with the same name already exists a prompt will appear asking to overwrite a previous match, if no this will add a number to the end of the filename.

Import Match

To import a match/matches go to the top left of the window and click the File menu button. Then click the Import Match button. This will open up a window to select a folder. Navigate to the relevant tournament folder and press ok. This will then import all files inside the tournament folder. If an error occurs the program will open an error prompt and the matches will not be imported. If a match with the same name already exists the program will add a number to the end of the imported match name.
